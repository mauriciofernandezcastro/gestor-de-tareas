Este es mi resumen profe:

Yo hice un proyecto que se llama TaskFlow. Es una página web bien simple para anotar tareas del día a día.

Lo que hice fue esto:

Hice 2 clases, una que se llama Tarea que guarda el texto, la fecha y si está lista, y otra que se llama GestorTareas que guarda todas las tareas en una lista y las puede borrar.

Traté de usar JavaScript más moderno, ya no usé var, usé solo let y const, y usé funciones flecha y esas comillas raras para meter texto al HTML.

Hice un formulario donde uno escribe la tarea y la agrega con un botón. También le puse que cambie de color cuando paso el mouse por encima.

Le puse unas cosas que se demoran, como que la tarea aparece después de medio segundo, un cartel que se borra solo a los 2 segundos y un contador que baja del 10 al 0.

Para que no se borren las tareas guardé todo en el localStorage del navegador. Y también hice que si no hay tareas, traiga unas de prueba de internet con fetch.

Para revisarlo solo hay que abrir el index.html en el navegador, no hay que instalar nada.

Espero que funcione y que me alcance para el 4.
